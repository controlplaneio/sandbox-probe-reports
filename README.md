# Sandbox Probe Reports

This repository houses [Sandbox Probe][sandbox-probe] reports from the daily scans.

This data can be used to analyze the changes in sandboxing over time.

Current reports tracked:

* `gemini`
  * [`sandbox_container`](./gemini/sandbox_container)
    * Specifically running the probe within a running `gemini-cli/sandbox` image

More scans and scan types will be added going forward.

[sandbox-probe]: https://github.com/controlplaneio/sandbox-probe
